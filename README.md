# Study in Lecture - C++ Practice

## 📌 Overview

This repository contains my C++ practice codes developed during lectures.
It focuses on fundamental **Object-Oriented Programming (OOP)** concepts and basic C++ implementations.

* Classes and Objects
* Constructors and Destructors
* Encapsulation
* Exception Handling
